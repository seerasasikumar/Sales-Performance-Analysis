# Power BI Dashboard

## Overview

This dashboard provides interactive insights into Superstore sales performance using Microsoft Power BI.

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Average Order Value
- Profit Margin

## Visualizations

- Sales by Category
- Profit by Category
- Sales by Region
- Profit by Region
- Top 10 Products
- Top 10 Customers

## Filters

- Region
- Category

## Key Business Insights

- Technology generates the highest sales.
- West region contributes the highest revenue.
- Furniture has the lowest profit.
- Top customers contribute a significant share of total sales.
