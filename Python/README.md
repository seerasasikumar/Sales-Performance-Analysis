# Python Data Analysis

## Overview

This folder contains Python scripts and notebooks used for Exploratory Data Analysis (EDA) on the Superstore Sales dataset.

## Libraries Used

- Pandas
- NumPy
- Matplotlib

## Analysis Performed

- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- KPI Calculation
- Sales Analysis
- Customer Analysis
- Product Analysis
- Monthly Sales Trend
- Profit Analysis
- Data Visualization

## Charts

- Sales by Category
- Monthly Sales Trend
- Top Products
